# 🇺🇸 English

## Flappy Bird - Advanced PyQt6 Edition

A modern and visually enhanced Flappy Bird game built with Python and PyQt6. This project includes animated graphics, particle effects, multiple themes, multilingual support, responsive UI, score tracking, lives system, and smooth gameplay.

### Features

- Modern PyQt6 graphical interface
- Smooth animations and particle effects
- Dynamic cloud and star background
- Dark and Light themes
- Multilingual support:
  - English
  - فارسی
  - 中文
- Score and Best Score tracking
- Lives system
- Pause and Resume support
- Responsive window resizing
- Keyboard and mouse controls
- FPS counter
- Modern game panel and HUD

---

## Requirements

- Python 3.10 or newer
- PyQt6

---

## Installation

### Install Required Libraries

```bash
pip install PyQt6
```

Or:

```bash
python -m pip install PyQt6
```

---

## Running the Project

```bash
python "Flappy Bird.py"
```

## Flappy Bird - 高级 PyQt6 版本

这是一个使用 Python 和 PyQt6 开发的现代化 Flappy Bird 游戏。项目包含丰富动画效果、粒子系统、多语言支持、主题切换、生命系统以及流畅的游戏体验。

### 功能特点

- 现代化 PyQt6 图形界面
- 流畅动画与粒子效果
- 动态云层与星空背景
- 深色与浅色主题
- 多语言支持：
  - English
  - فارسی
  - 中文
- 分数与最高纪录系统
- 生命值系统
- 暂停与继续功能
- 自适应窗口大小
- 键盘和鼠标控制
- FPS 显示
- 现代化控制面板
